
# Installation

```sh
./setup/gcp-create.sh
./setup/gcp-link.py
```

# Starting 

## GCP

```sh
./setup/gcp-start.sh
./setup/gcp-link.py
```

## TensorSpdz

```sh
./start.sh
```

# Stopping

## GCP

```sh
./setup/gcp-stop.sh
```

## TensorSpdz

```sh
./stop.sh
```

# Cleanup

```sh
./setup/gcp-destroy.sh
```

see https://cloud.google.com/solutions/running-distributed-tensorflow-on-compute-engine
